# rar and unrar

To fix a corrupted file:
```bash
$ rar r file.rar
```

To test a file (or sequence of files):
```bash
$ unrar t file[1].rar
```

To unrar with directory hierarchy:
```bash
$ unrar x file.rar
```

To unrar without directory hierarchy (i don't use this ever):
```bash
$ unrar e file.rar
```
