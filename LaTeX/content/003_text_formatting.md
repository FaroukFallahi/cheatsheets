# Text formatting

Bold text:

```latex
\textbf{this is bold}
```

Italic text:

```latex
\emph{this is italic}
```

Underlined text:

```latex
\underline{this is underlined}
```

Non-indented paragraph:

```latex
Here is a paragraph

\noindent This is a non-indented paragraph.

And another paragraph
```

Single and double quotes:

```latex
This is `single' and this is ``double" quote!
```
